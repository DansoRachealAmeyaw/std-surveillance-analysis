# A Decade of STD Trends: From Data to Action

![STD Surveillance Dashboard](images/std-surveillance-dashboard.png)

## Project Overview

This project examines global sexually transmitted disease surveillance trends from 2015 to 2025. The goal was to identify important patterns in reported cases and mortality, compare regions and countries, and translate the findings into practical recommendations for public-health stakeholders.

## Tools Used

* Microsoft Excel for data cleaning and exploratory analysis
* SQL for querying and aggregating the dataset
* Power BI for dashboard development
* PowerPoint for presenting the findings

## Key Questions

1. How did reported STD cases change between 2015 and 2025?
2. Which diseases contributed most strongly to the overall burden?
3. Which regions and countries recorded the highest burden?
4. What insights can support targeted public-health interventions?

## Dashboard Highlights

The dashboard explores:

* global trends in reported STD cases;
* disease-burden composition;
* regional comparisons;
* country-level differences;
* Africa-focused analysis;
* a Ghana-specific deep dive;
* recommendations for screening, vaccination, treatment access, and surveillance improvement.

## Key Findings

* Reported STD cases fluctuated throughout the period without a sustained downward trend.
* A relatively small group of diseases contributed substantially to the overall burden.
* The burden was unevenly distributed across regions and countries.
* High-burden locations require targeted rather than uniform interventions.
* Continued investment in screening, vaccination, treatment access, and surveillance is necessary.

## Limitations

This project is descriptive and exploratory. The patterns identified should not be interpreted as proof of causality. Differences in surveillance quality, reporting systems, diagnostic access, population size, and healthcare infrastructure may influence the recorded values.
